  啟動 Windows 虛擬機器 
 
             
 
            
 
           
 
           
 
            
 
             
使用 Amazon EC2
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
在此教學中，您將學習如何使用Amazon Elastic Compute Cloud (EC2) 啟動、設定和連接到 Windows 虛擬機器。Amazon EC2 是一種可用來在雲端建立和執行虛擬機器的 Amazon Web Service (我們稱這些虛擬機器為「執行個體」)。此教學中的所有操作都可用免費方案資格來進行。
 
           
 
          
 
         
 
        
 
        
 
         
 
          
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
 
                  
在 AWS 中啟動虛擬機器需要帳戶
 
                  幾分鐘內建立免費帳戶 
                  
 
                  
 
                   
AWS 免費方案包括為期一年每月 750 小時的 Linux 和 Windows t2.micro 執行個體使用量。若要繼續符合免費方案資格，您只能使用 EC2 微型執行個體。
 
                   
查看 AWS 免費方案詳細資訊 »
 
 
                  
 
                 
 
                
 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
             
 
              
  步驟 1：進入 EC2 儀表板 
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
按一下這裡，即可在新的瀏覽器分頁開啟 AWS 管理主控台，以讓此教學持續保持開啟狀態。在 Compute 下尋找 EC2，然後按一下以開啟 Amazon EC2 主控台。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
             
 
              
  步驟 2：建立和設定您的虛擬機器 
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
a. 您現在位於 Amazon EC2 主控台。按一下 Launch Instance 
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
b. 使用 Amazon EC2 可以指定所要使用之執行個體的軟體與規格。這個畫面會顯示選擇 Amazon Machine Image (AMI) 的選項，這個範本包含啟動執行個體所需的軟體組態。 
 
                 
在本教學中，尋找 Microsoft Windows Server 2012 R2 Base，然後按一下 Select。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
c. 您現在將選擇執行個體類型。執行個體類型包括各種 CPU、記憶體、儲存和聯網容量的不同組合，所以您可以為應用程式選擇適當的組合。如需詳細資訊，請參閱 Amazon EC2 執行個體類型。
 
                 
在本教學中，選取預設選項 t2.micro – 此執行個體類型涵蓋在免費方案中。接著，按一下頁面底部的 Review and Launch。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
d. 您可以檢閱為執行個體選取的選項，包括 AMI 詳細資訊、執行個體類型、安全群組、執行個體詳細資訊、儲存和標籤。您可以保留這些選項的預設值，並按一下頁面底部的 Launch。
 
                 
注意：如需有關選項的詳細資訊，請參閱啟動執行個體
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
             
 
              
  步驟 3：建立金鑰對和啟動您的執行個體 
 
             
 
            
 
           
 
           
 
            
 
             
要連線到虛擬機器，您需要金鑰對。金鑰對是用來登入執行個體 (就像進入家裡所使用的家門鑰匙)。
 
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
a. 在彈出的視窗中，選取 Create a new key pair，然後命名為 MyFirstKey。接著，按一下 Download Key Pair。MyFirstKey.pem 將會下載到您的電腦，務必將此金鑰對儲存到安全的電腦位置。 
 
                 
     
                  
  • Windows 使用者：我們建議將您的金鑰對儲存在 user 目錄中名為 .ssh 的子目錄 (例如，C:\user\{yourusername}\.ssh\MyFirstKey.pem)。
    •  
                  
  • Mac/Linux 使用者：我們建議將您的金鑰對儲存在主目錄的 .ssh 子目錄中 (例如，~/.ssh/MyFirstKey.pem)。
    •  
                 
 
                 
注意：如果您不記得 SSH 私有金鑰 (您下載的檔案) 的存放位置，就無法連接虛擬機器。
 
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
b. 下載並儲存金鑰對之後，按一下 Launch Instance 啟動您的 Windows Server 執行個體。
 
                 
注意：啟動執行個體需要幾分鐘的時間。
 
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
c. 在下個畫面按一下 View Instances，檢視您剛建立的執行個體並查看其狀態。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
             
 
              
  步驟 4：連接到您的執行個體 
 
             
 
            
 
           
 
           
 
            
 
             
啟動執行個體之後，接下來便是擷取管理員密碼並使用 Remote Desktop Protocol (RDP) 用戶端連接它。
 
             
如果您需要，AWS 文件包含有關如何安裝 RDP 用戶端的資訊。 
 
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
a. 選取您剛建立的 Windows Server 執行個體，然後按一下 Connect
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
b. 若要連線到您的 Windows 虛擬機器執行個體，需要使用者名稱和密碼：
 
                 
     
                  
  • 使用者名稱預設值為 Administrator
    •  
                  
  • 若要收到您的密碼，按一下 Get Password
    •  
                 
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
c. 若要擷取密碼，您需要找到在步驟 3 建立的金鑰對。按一下 Choose File，然後瀏覽到您存放 MyFirstKey.pem  的目錄。您的金鑰對將會出現在文字方塊中。按一下 Decrypt Password。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
d. 您現在擁有 Windows Server 執行個體的加密密碼。務必將此資訊儲存在安全的位置。這是您的 Windows Server 管理員登入資料。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
e. 按一下 Download Remote Desktop File 並開啟檔案。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
f. 提示您登入執行個體時，使用產生的使用者名稱和密碼連接到您的虛擬機器。
 
                 
注意：完成此步驟之後，您可能會收到無法驗證安全憑證的警告。只要選取 yes，然後繼續完成連接到 Windows Server 執行個體即可
 
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
             
 
              
  步驟 5：終止您的 Windows VM 
 
             
 
            
 
           
 
           
 
            
 
             
您可以從 Amazon EC2 主控台輕鬆終止 Windows Server VM。事實上，終止不再使用的執行個體是一項最佳實務，如此您就不需要繼續為它們付費。
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
a. 返回 EC2 主控台，選擇所建立之執行個體旁邊的方塊。然後按一下 Actions 按鈕，導覽至 Instante State 再按一下 Terminate。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
           
 
           
 
            
 
             
 
              
 
               
 
                
 
                 
b. 系統將要求您確認終止 – 選擇 Yes, Terminate.
 
                 
注意：此程序會花幾秒鐘的時間來完成。一旦您的執行個體終止之後，EC2 主控台上的 Instance State 會變成 terminated。
 
                
 
               
 
              
 
             
 
             
 
              
 
               
 
                 
                
 
                 
 
                  
 
                    
                  
 
                 
 
                
 
                 
               
 
              
 
             
 
            
 
           
 
          
 
         
 
        
 
       
 
      
 
      
 
       
 
        
 
         
 
          
 
           
 
            
 
             
 
            
 
           
 
           
 
            
 
             
 
              
  恭喜您！ 
 
             
 
            
 
           
 
           
 
            
 
             
您已經使用 Amazon EC2 建立、設定和連接到雲端中的第一個 Windows 虛擬機器。無須購買伺服器或管理資料中心的成本和複雜過程，您可以將更多的資源投注到解決方案。
